Github Integration for Selenium Builder


A plugin for Selenium Builder that lets you store your Selenium scripts on GitHub. You can browse, edit, and re-commit test scripts.

You can install GitHub Integration directly from Selenium Builder by selecting "Manage Plugins" in the startup options.

GitHub Integration for Selenium Builder was written by David Stark at the behest of Sauce Labs. It is licensed under the Apache 2 licence. It is not associated with GitHub in any way.
